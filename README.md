## About The Project

Car Leasing Calculator made to help customers with their monthly leasing costs. Dynamicly calculates the costs while the user is entering the data.

## Getting Started

Currently the application runs locally so you need to download it on your computer or simply clone the repository.


### Built with

-  HTML
-  CSS
-  JavaScript

<summary>File Structure</summary>
   <ul>    
    <li><i>Index.html</i> - default entry point for a website. It contains the basic structure of the web page, including elements such as the head (which includes metadata, title, and links to CSS or scripts) and the body (the main content of the page). This file is essential for defining how the website appears and functions when accessed by users.</li>
    <li><i>app.js</i> - JavaScript file containing the main application logic. It handles functionalities such as user interactions, data manipulation. This file is crucial for adding interactivity and dynamic behavior to the website.</li>
    <li><i>styles.css</i> - CSS file that holds all the style rules. There you can find how a certain element is displayed and change the element's properties. The responsive design is made via @media. The @media rule is used in media queries to apply different styles for different media types/devices.</li>
    <li><i>calculator.ico</i> - icon shown in browser's tab.</li>
  </ul>
